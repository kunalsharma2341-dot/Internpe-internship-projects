# DayList — Responsive To-Do List App

DayList is Task 3 of my web development internship. It is a responsive task-management application built using HTML, CSS and JavaScript.

## Features

- Add, edit, complete and delete tasks
- Task categories, priorities and optional due dates
- Search and filter tasks
- Sort by newest, due date or priority
- Overdue-task indication
- Progress tracking
- Clear all completed tasks
- Light and dark themes
- Tasks saved using browser `localStorage`
- Responsive mobile, tablet and desktop layout
- Accessible labels and keyboard-friendly controls

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Run in VS Code

1. Open the project folder in Visual Studio Code.
2. Open `index.html`.
3. Right-click and select **Open with Live Server**.

You may also open `index.html` directly in a modern web browser.

## How Storage Works

Tasks and the selected colour theme are stored locally in the browser. No account, server or database is required. Clearing the browser's site storage will remove saved tasks.
