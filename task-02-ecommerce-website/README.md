# ShopNest E-commerce Website

ShopNest is a responsive front-end e-commerce project created with HTML, CSS and JavaScript for a web development internship task.

## Features

- Responsive desktop, tablet and mobile layout
- Product images stored locally in the `assets` folder
- Product search and category filters
- Add-to-cart and quantity controls
- Cart saved with browser `localStorage`
- Everyday Kit bundle logic: 10% off any three different products
- Demo checkout form with validation
- Accessible mobile navigation

## Run in VS Code

1. Extract this project folder.
2. Open Visual Studio Code.
3. Select **File → Open Folder** and choose `ShopNest-Ecommerce-VSCode`.
4. Open `index.html`.
5. Right-click inside the file and select **Open with Live Server**.

If Live Server is not installed, install the extension named **Live Server** by Ritwick Dey. You can also open `index.html` directly in a browser.
