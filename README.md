# InternPe Web Development Internship Projects

This repository contains the web development projects I completed during my internship at InternPe.

The projects are built using HTML, CSS and JavaScript. Through these tasks, I practised responsive web design, DOM manipulation, JavaScript logic, event handling and creating interactive browser-based applications.

## Projects

### Task 1 – Simple Calculator

A basic calculator that performs common arithmetic operations through a clean and responsive interface.

**Technologies:** HTML, CSS and JavaScript

### Task 2 – E-commerce Website

A responsive e-commerce website featuring product listings, category filtering and an interactive shopping interface.

**Technologies:** HTML, CSS and JavaScript

### Task 3 – To-Do List App

A simple task-management application that allows users to add, complete and remove daily tasks.

**Technologies:** HTML, CSS and JavaScript

### Task 4 – Connect Four Game

A two-player browser game with custom player names, score tracking, multiple rounds, move undo and win detection.

**Technologies:** HTML, CSS and JavaScript

## What I Learned

While completing these projects, I improved my understanding of:

- Structuring webpages with semantic HTML
- Creating responsive layouts using CSS
- Handling user interactions with JavaScript
- DOM manipulation and event listeners
- Arrays, functions and conditional logic
- Organising projects and documentation on GitHub
