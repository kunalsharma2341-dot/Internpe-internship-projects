# InternPe Web Development Internship Projects

This repository contains the projects I completed during my web development internship at **InternPe**.

The projects are built using HTML, CSS and JavaScript. Each task helped me practise responsive design, JavaScript logic, browser storage, debugging, project organisation and deployment with GitHub Pages.

## Task 1 – Smart Calculator

Task 1 was to create a functional and responsive calculator using HTML, CSS and JavaScript.

### Features

- Addition, subtraction, multiplication and division
- Percentage calculations
- Decimal-number support
- Clear and delete controls
- Positive and negative number support
- Division-by-zero error handling
- Keyboard input support
- Responsive mobile and desktop design

### What I Learned

- Handling button and keyboard events
- Storing numbers and mathematical operators
- Updating webpage content using JavaScript
- Validating calculations and handling errors
- Separating HTML, CSS and JavaScript files

## Task 2 – Responsive E-commerce Website

Task 2 was to build a responsive front-end e-commerce experience with product discovery and shopping-cart functionality.

### Features

- Responsive mobile, tablet and desktop layout
- Product search
- Category filters
- Shopping cart with quantity controls
- Cart persistence using `localStorage`
- Optimised WebP product images
- Demo checkout form with validation
- Accessible mobile navigation
- Custom **Everyday Kit** offer that applies a 10% discount after three different products are added

### Project Links

- [Live Demo](https://kunalsharma2341-dot.github.io/Internpe-internship-projects/task-02-ecommerce-website/)
- [View Source Code](task-02-ecommerce-website/)

### What I Learned

- Rendering product information dynamically
- Searching and filtering JavaScript data
- Managing shopping-cart state
- Saving data with `localStorage`
- Debugging duplicate interface elements
- Organising product images and relative file paths
- Deploying a project inside a GitHub Pages subfolder

## Task 3 – Responsive To-Do List App

Task 3 was to create a responsive task-management application using HTML, CSS and JavaScript.

### Features

- Add, edit, complete and delete tasks
- Task categories and priority levels
- Optional due dates
- Overdue-task indication
- Search functionality
- Active and completed filters
- Sorting by newest, due date or priority
- Progress tracking
- Clear completed tasks
- Light and dark themes
- Task persistence using `localStorage`
- Responsive mobile, tablet and desktop layout
- Accessible labels and keyboard-friendly controls

### Project Links

- [Live Demo](https://kunalsharma2341-dot.github.io/Internpe-internship-projects/task-03-todo-list-app/)
- [View Source Code](task-03-todo-list-app/)

### What I Learned

- Creating and updating task objects
- Dynamically rendering lists with the DOM
- Combining search, filter and sorting logic
- Saving structured data in the browser
- Editing existing records
- Managing responsive layouts and colour themes
- Handling empty states and user feedback

## Running the Projects Locally

1. Download or clone this repository.
2. Open the repository folder in Visual Studio Code.
3. Open the folder of the project you want to run.
4. Open its `index.html` file with Live Server.

You can also open an `index.html` file directly in a modern browser.
