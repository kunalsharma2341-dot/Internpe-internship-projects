How to play

  1. Players take turns dropping one disc into a column.
  2. The disc falls into the lowest empty space.
  3. Connect four discs horizontally, vertically or diagonally to win.

About this project

This project uses two-dimensional arrays, DOM events and game-state logic. It was created to practise front-end development through a small interactive application.

Developed by Kunal Sharma as a web development internship project.
